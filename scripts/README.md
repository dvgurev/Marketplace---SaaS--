# Scripts

Утилиты и скрипты автоматизации: сборка, тесты, миграции, бэкапы,
управление контейнерами.

Скрипты могут быть на bash, Python или другом языке. Они служат для ускорения
рутинных задач и часто используются в CI/CD пайплайне.

```
scripts/
├── build.sh
├── migrate.sh
├── backup.sh
└── README.md
```
