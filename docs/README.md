# Docs

Документация проекта: техническое задание, архитектурные решения, руководства
разработчика и пользователя.

Формат может быть Markdown, AsciiDoc или сгенерированные HTML/Confluence-страницы.

## Содержимое

```
docs/
├── architecture.md      # описание архитектуры
├── api.md               # спецификация REST/GraphQL API
├── dev-guide.md         # инструкции для разработчиков
├── user-guide.md        # руководство по работе с платформой
└── README.md
```
