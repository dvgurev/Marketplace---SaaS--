# src

Исходный код оркестратора (backend) платформы.

## Структура

- `index.ts` — точка входа приложения.
- `config.ts` — загрузка переменных окружения и констант.
- `prisma.ts` — инстанс Prisma Client.
- `routes/` — HTTP API модули.
- `middleware/` — Express middleware (авторизация и др.).
- `utils/` — вспомогательные функции (хеширование, JWT и т.п.).

## Назначение

Здесь разворачивается бизнес-логика: обработка запросов, работа с БД через Prisma,
интеграции и т.д. В большинстве случаев добавляйте новые сущности через
новый маршрут в папке `routes` и соответствующие утилиты.
