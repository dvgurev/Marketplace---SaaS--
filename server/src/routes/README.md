# routes

Здесь находятся модули маршрутов Express, которые организуют HTTP API оркестратора.
Каждый файл экспортирует `Router` с приватными и публичными эндпоинтами.

## Содержимое

- `auth.ts` — регистрация / аутентификация
- `users.ts` — управление пользователями
- `services.ts` — CRUD сервисов, доступных на платформе
- `servicePlans.ts` — тарифные планы для сервисов
- `subscriptions.ts` — логика подписок пользователей
- `tenants.ts` — информация о развернутых контейнерах
- `payments.ts` — записи платежей

## Добавление нового маршрута

1. создайте файл в этой директории;
2. реализуйте логику с `authenticate` при необходимости;
3. монтируйте роутер в `src/index.ts`.
