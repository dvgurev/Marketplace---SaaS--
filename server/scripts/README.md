# scripts

Утилиты и вспомогательные скрипты для сервера.

## Структура

- `seed.ts` — создание или обновление учётной записи администратора.
- в будущем сюда могут добавляться миграционные, бэкапные скрипты и т.п.

## Назначение

Запуск производится через npm-скрипты:

```bash
npm run seed -- <email> <password>
```

или напрямую: `ts-node-dev --transpile-only ./scripts/seed.ts`.
