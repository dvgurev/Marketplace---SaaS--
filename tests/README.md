# Tests

Набор автоматических тестов по всему проекту: unit, интеграционные, e2e.

```
tests/
├── client/            # тесты фронтенда (Jest, Cypress и т.п.)
├── server/            # тесты бэкенда (Go/Python/Node.js)
└── services/          # тесты отдельных сервисов
```

## Запуск всех тестов

```
cd tests && ./run_all.sh
```
