# Client

Фронтенд-приложение маркетплейса SaaS-сервисов.

## Описание

Этот каталог содержит исходники пользовательской части платформы, реализованной на современном
фреймворке (React, Vue, Angular и т.п.). Здесь находятся компоненты интерфейса, страницы,
стили, конфигурация сборки и тесты.

## Структура (пример)

```
client/
├── public/            # статические ресурсы (index.html, favicons)
├── src/
│   ├── components/    # переиспользуемые UI-компоненты
│   ├── pages/         # страницы приложения
│   ├── hooks/         # кастомные хуки или утилиты
│   ├── services/      # клиентские API-обёртки
│   ├── store/         # состояние (Redux/Vuex/MobX и т.п.)
│   └── styles/        # глобальные стили
├── tests/             # unit/integraционные тесты
├── package.json
└── README.md
```

## Запуск

```
cd client
npm install
npm start
```

## Тесты

```
npm test
```
